TYPE: course

COURSE: CS107
TITLE: Computer Organization and Systems

LEVEL: undergraduate

DESCRIPTION:
Introduction to the fundamental concepts of computer systems. Explores how computer systems execute programs, store information, and communicate. Topics include C and assembly language programming, translation of code to machine language, machine-level data representation, memory hierarchy and caching, processes and threads, linking, and system-level I/O. Students gain a deep understanding of the gap between high-level language code and the actual execution on hardware.
