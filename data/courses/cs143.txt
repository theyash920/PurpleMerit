TYPE: course

COURSE: CS143
TITLE: Compilers

LEVEL: undergraduate

DESCRIPTION:
Principles and practices for design and implementation of compilers and interpreters. Topics include lexical analysis, parsing theory, type checking, code generation and optimization. Students will build a working compiler for a substantial programming language as a course project. Covers formal language theory as applied to compiler construction, including regular expressions, context-free grammars, and attribute grammars.
